Aim of this project is to enable reservoir computing for .NET without any other dependencies.

Currently implemented:
	ESN (Echo State Network)
	FF/BasicNetwork (Feed Forward Network)
	LinRegrTrainer (Linear Regression)
	RPROPTrainer (Resilient Propagation)

Usage is demonstrated in demo console application (/Demo/ESNDemoConsoleApp)
